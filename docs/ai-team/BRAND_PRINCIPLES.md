# BRAND_PRINCIPLES — ブランド原則

## 憲法
常に: Premium / Timeless / Intelligent / Calm / Curious / Human / Minimal / Research-driven。
犠牲にしない: Accessibility / Performance / Readability / Maintainability。
衝突時の優先順位: 安全 → アクセシビリティ → 正しさ → ユーザー理解 → パフォーマンス → 保守性 → ブランド一貫性 → 事業価値 → 視覚的洗練 → 新規性。

## コアコピー
- 「無駄をなくして、熱中を取り戻す。」 / "Less friction. More you."
- 売り方は「AIで急げ」ではなく「無駄を減らして熱中・余白を取り戻す」

## 声
- 「一人の思索する書き手」。営業感・誇大数字に極めて敏感なブランド
- 数字は約束しない: 幅表示（30〜50%）＋仮定明記＋観察形
- コンサル用語（棚卸し・導線・定着・見える化・まるごと担当）→ 書き手の語彙（書き出す・洗い出す・分ける・整える）
- 脅し文法禁止（「あなたの夜を奪っています」→ 観察形へ書き換え済み）
- 英語ラベルは品位重視: SIGNS / ROOM / TOOLS WE THINK WITH / Pricing

## ビジュアル
- 配色: クリーム `#f0e6d6` / ネイビー `#0d2030` / サーモン `#e8866a`（+ `--accent-ink #7a4025` 小文字用）
- 書体: 見出し=Noto Serif JP（300/500, palt詰め）、本文=Zen Kaku Gothic New、ラベル=JetBrains Mono
- 罫線 .5px、影は最小限。hoverの跳ね上げ（translateY）禁止 — 罫線・影の変化のみ（Stripe Press的な静かな応答）
- 参照ベンチマーク: iA / Stripe Press / Apple HIG / MIT Media Lab / 北欧、暮らしの道具店

## 演出の判断
- 却下済みパターン: タイプライター・ライブ時計・ティッカー・カスタムカーソル・磁石ボタン・スクロールハイジャック・ニューラルパーティクル
- 人間味の担い手: console の手紙・コナミコマンド・ソースコメント（これらは維持する）
- ページごとの主役インタラクションは1つまで
