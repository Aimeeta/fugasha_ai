# 調査報告：ミニマル/上質さを保ったまま「遊び心・人間味・私らしさ」を出す手法カタログ（Webリサーチ）

検証メモ: 確認済み＝WebFetchで実際にアクセスし内容確認。cassie.codes は現在閉鎖挨拶ページのみ。heystudio.es は接続拒否で未確認。CSSレベルの記述は実例ソース直読ではなく一般技法（技法解説記事は確認済み）。

## A. すぐ真似できる小技（工数小：数時間〜1日）

1. **マーカー風ハイライト（蛍光ペンの跡）** — `background-image: linear-gradient` を `<mark>` に敷き、border-radius を四隅で不揃いに、`box-decoration-break: clone`。色はアクセント低彩度版。純CSS・a11y影響なし。1画面1〜2箇所が上限。（技法記事: dev.to, max.hn 確認済み）
2. **自筆署名・手書きの一言（SVGパス化）** — 実際にAimeeが書いた署名や「読んでくれてありがとう」をベクタ化→インラインSVG。手書きWebフォントは不採用が正しい（フォント規則と衝突）。筆跡は複製不能な身体の索引。**「AIで作った感」への最直接の解毒剤。**（参考文脈: ほぼ日, Craig Mod）
3. **写真の微回転・ポラロイド風配置** — `transform: rotate(±2deg以内)`、白フチ、柔らかい影1枚、キャプション。複数枚なら回転角を不揃いに。静的なのでreduced-motion配慮不要。
4. **サイドノート／脚注（欄外注）** — Tufte CSS（確認済み: edwardtufte.github.io/tufte-css）の `label.margin-toggle`＋checkbox＋`span.sidenote` 3点セット。広い画面では本文右余白に浮かび、狭い画面ではタップ開閉。JSゼロ・キーボード可。**記事ページに最も推奨。**
5. **「now」セクション** — Derek Sivers /now（確認済み: sive.rs/now）、nownownow.com。「2026年7月、〇〇の検証をしています」程度。サイトが生きている証拠。更新が止まると逆効果なので四半期1回の粒度で。
6. **日付＋場所のスタンプ** — 記事末尾に「2026年7月・自宅の机から」を --font-mono の小さな文字で。手紙の作法。
7. **footerの一言（正直な小声）** — Piccalilli（確認済み）の立場表明のように、「このサイトは私が一人で、手で書いています」「外部JSはスクロール用の1つだけ」等の検証可能な事実の一行。
8. **オリジナル404ページ** — GitHub Pagesはルートの404.htmlを自動使用。「道に迷いましたね。お茶でもどうぞ」＋静かな導線。見えないところまで丁寧の証明。
9. **セクション名・コーナー名の固有名詞化** — ほぼ日（確認済み）の「言いまつがい」文化。「記事一覧」→「書きもの」「観察ノート」など風雅舎の語彙（書き出す・洗い出す・整える）で命名し直す。コピー変更のみで低リスク高効果。
10. **等身大の一行（自己開示・軽い自虐）** — Craig Mod "some total nudnik"、Maggie Appleton "mediocre developer"（確認済み）。「Excelの関数はいまだに毎回調べます」のような一行。弱みの開示は信頼の通貨。
11. **エピグラフ（章頭の引用）** — Tufte CSS の blockquote+footer 型。Noto Serif JP 300 のやや大きめで短い引用＋出典。読書遍歴の露出＝Curious の証明。
12. **記事タイトルの「好奇心フック」化** — Marcin Wichary / aresluna.org（確認済み）。「AI導入のポイント5選」型をやめ「請求書の宛名を直すのに40分かかっていた話」のような固有の観察から始まるタイトルへ。
13. **「道具箱」ページ（/uses文化）** — 「私が実際に使っている道具」一覧を一言コメント付きで。道具の選択は身体性と好みの露出。

## B. 構造的な改修（工数大：数日〜）

14. **エディトリアルな「レイアウトの緩急」** — Stripe Press（確認済み）はセクションごとに組みの方向を変えリズムを作る。日本デザインセンター（確認済み）は大量の余白と短い見出しで「間」を作る。①1カラム大組み→②2カラム非対称(3:7)→③余白だけの間奏→④密度の高いリスト、と密→疎→密の呼吸を設計。**「AIで作った感＝単調」への最も本質的な回答**（小技の総和より効く）。
15. **番号付き章立て＋静かな目次** — セクション見出しに mono の 01/02 を添え、ページ冒頭に目次。IntersectionObserverで現在地表示可（既存方針内）。
16. **デジタルガーデン（記事に育ち具合メタデータ）** — Maggie Appleton（確認済み）の Essays/Notes/Patterns ＋経過時間表示。「芽・育成中・手入れ済み」ラベル＋最終手入れ日。更新運用の覚悟が必要。
17. **連載もの・定型の小さな楽しみ** — 北欧、暮らしの道具店（確認済み）の4コマ連載、ほぼ日の毎日コラム。「今月の観察」「AIに頼んで失敗したこと月報」など同じ器で繰り返す連載枠を1つ。
18. **「仕事の裏側」ジャーナル** — 土屋鞄（確認済み）のジャーナル、Craig Modの散らかった仕事場写真。「このサイト自体をどう作っているか」を記事化＋作業机の写真1枚（#3と連動）。
19. **マニフェスト／価値観ページ** — Rauno Freiberg（確認済み: rauno.me）の7行マニフェスト、homesickdesign（確認済み・盛岡）は東北の不利を率直に書く。サイト憲法を顧客向けの言葉に翻訳して公開。「やらないことリスト」（誇大な数字を約束しない等）は特に強い。
20. **年次アーカイブ** — Rauno Freibergは2022/2023年版サイトをアーカイブとして残す。大改修時に旧版を /2026/ 等に静的保存しfooterから小さくリンク。成長の年輪。
21. **スクロール途中の「間奏」セクション** — 日本デザインセンター（確認済み）の瞑想的短文＋大余白。一行だけのセクション（vh高め・文字小さく中央）。沈黙の設計。Calmのまま個性を出す最上級の手法。
22. **カーソル演出・音は非推奨** — Josh W. Comeau（確認済み）は音・イラストで成立するが彼のブランドは「賑やかで楽しい先生」。風雅舎の声量では「署名・欄外注・footerの小声・404」が適量。借りるなら「ライト/ダークで差し替わる自画像イラスト」（インラインSVG）と「本人の声で書く」姿勢のみ。

## 優先順位の提案
